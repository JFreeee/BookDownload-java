# 📚 BookDownload
一个基于 Spring Boot + Vue 的电子书下载网站

---
## 🚀 项目功能
- 用户注册 / 登录
- 上传电子书（PDF + 封面）
- 图书搜索
- 管理员后台
---

## 🛠 技术栈
- 后端：Spring Boot + MyBatis
- 前端：Vue2 + Axios + Bootstrap
- 数据库：PostgreSQL（Supabase）
- 文件存储：Cloudinary
- 部署：
  - 前端：Vercel
  - 后端：Render
---

## 🌐 在线地址
- 前端：https://book-download-java.vercel.app
- 后端：https://bookdownload-java.onrender.com
---

## ⚠️ 注意
由于 Render 免费服务存在冷启动机制，首次访问后端可能需要约 1 分钟时间启动。
请耐心等待后刷新页面。
